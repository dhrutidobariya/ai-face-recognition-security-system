# AI Face Recognition WhatsApp Alert System

This project is an AI-based security system that detects authorized and unauthorized persons using face recognition and sends real-time WhatsApp alerts with the captured image.

## 🚀 Features

- Face detection using DeepFace
- Authorized person verification
- Unauthorized person alert system
- Automatic image capture using webcam
- WhatsApp notification with captured image
- Cloud image hosting using ImgBB

## 🧠 Technologies Used

- Python
- OpenCV
- DeepFace
- Twilio WhatsApp API
- ImgBB API


## 🔐 Environment Variables

Create a `.env` file and add your credentials:
- TWILIO_SID=your_twilio_sid
- TWILIO_TOKEN=your_twilio_token
- IMGBB_KEY=your_imgbb_api_key
- WHATSAPP_NUMBER=whatsapp:+91xxxxxxxxxx
